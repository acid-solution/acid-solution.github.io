### 每日内容总结
这周主要是把 Go 的基础语法和后端开发中最常见的几个概念过了一遍。

06/01 先把 Go 环境跑通了，完成了 `go mod init`、`go run .`、`go build .` 这些最基本的项目创建、运行和编译流程，并创建了一个 hello 项目。这个阶段主要是把后面能继续写代码的环境准备好。

06/02 学了变量、函数和包。变量部分主要理解了 Go 是强类型语言，但可以自动推断类型，也有零值、短变量声明 `:=`、声明后必须使用等规则。函数部分接触了多返回值和错误返回。包这块理解了 `main` 包用于运行，其他包用于被调用；同一个文件夹下的 Go 文件必须属于同一个包；不同包之间调用需要 `import`，而且大写开头的标识符才能被其他包访问。

06/03 学了 slice、map 和 range。slice 这块主要理解了可变列表、`append`、`make`、切片截取等用法，尤其是 `append` 会返回新的 slice，所以需要重新赋值。map 这块学了键值对、`value, ok := mp[key]`、`delete`、nil map 不能直接写入，以及 map 遍历顺序不固定。range 则是 Go 里遍历 slice 和 map 的常用写法，同时也注意到 range 里的 value 是副本。

06/04 学了 struct、method 和 receiver。struct 更适合表示固定字段的数据，map 更适合动态键值对。Go 没有传统 Java 那种 class，所以 struct 在组织数据时很重要。receiver 可以把函数绑定到类型上，形成方法；值 receiver 拿到的是副本，指针 receiver 可以修改原数据。现在对 method 和 receiver 有了基本感觉，但它们和 interface 的关系还需要后面继续学。

06/05 学了 error handling。Go 的错误不是 try-catch，而是普通返回值，所以每一层都要显式处理。也学到了用 `fmt.Errorf("...: %w", err)` 给错误加上下文，用 `errors.Is` 判断固定错误类型，用 `errors.As` 取出自定义错误里的具体信息。这里开始能感觉到 Go 的错误处理虽然啰嗦，但逻辑会比较清楚。

06/06 做了一个 Go 综合练习：用户积分统计系统。这个练习把前面学到的 struct、map、函数和错误处理串到了一起，用来处理用户不存在、积分非法等情况。相比单独看语法，这种小练习更能暴露哪些地方是真的会了，哪些地方只是看懂了。

## 本周GO综合练习

### 练习要求
用户积分统计系统

目标：写一个简单程序，统计用户积分，并处理用户不存在、积分非法等错误。

### 全部代码
```go
package main

import (
	"errors"
	"fmt"
)

// 定义四种错误变量
var ErrInvalidUser = errors.New("invalid user")
var ErrUserAlreadyExists = errors.New("user already exists")
var ErrUserNotFound = errors.New("user not found")
var ErrInvalidScore = errors.New("invalid score")

// 定义用户结构体
type User struct {
	ID    int64
	Name  string
	Score int
}

// 定义用户管理器
type UserStore struct {
	users map[int64]*User
}

// 用户管理器的构造函数
func NewUserStore() *UserStore {
	return &UserStore{
		users: make(map[int64]*User),
	}
}

// 添加用户
func (s *UserStore) AddUser(user *User) error {
	if user == nil {
		return ErrInvalidUser
	} else if _, exists := s.users[user.ID]; exists {
		return ErrUserAlreadyExists
	}

	s.users[user.ID] = user
	return nil
}

// 根据id查用户
func (s *UserStore) GetUser(id int64) (*User, error) {
	if user, exists := s.users[id]; exists {
		return user, nil
	} else {
		return nil, ErrUserNotFound
	}
}

// 给用户加积分
func (s *UserStore) AddScore(id int64, score int) error {
	if score < 0 {
		return ErrInvalidScore
	}

	user, err := s.GetUser(id)
	if err != nil {
		return err
	}
	user.Score += score
	return nil
}

func main() {
	//创建用户管理器
	store := NewUserStore()

	err := store.AddUser(&User{
		ID:    1,
		Name:  "Tom",
		Score: 10,
	})
	if err != nil {
		fmt.Println("add user failed:", err)
		return
	}

	err = store.AddScore(1, 20)
	if err != nil {
		fmt.Println("add score failed:", err)
		return
	}

	user, err := store.GetUser(1)
	if err != nil {
		fmt.Println("get user failed:", err)
		return
	}

	fmt.Println(user.Name, user.Score)
}
```
运行结果：

```text
Tom 30
```
